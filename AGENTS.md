# AGENTS.md

## 目的
AI エージェント共通の作業方針を定義します。

## 基本方針
- 会話言語: 日本語
- コメント言語: 日本語
- エラーメッセージ言語: 英語
- 日本語と英数字の間には半角スペースを挿入する
- コミットメッセージ: Conventional Commits に従う（description は日本語）

## 判断記録のルール
1. 判断内容の要約
2. 検討した代替案
3. 採用理由と不採用理由
4. 前提条件・仮定・不確実性の明示

## 開発手順
1. プロジェクトの理解（`package.json` や `src/` の確認）
2. 依存関係のインストール（`pnpm install`）
3. 実装・修正
4. Lint の実行（`pnpm lint`）
5. フォーマットの実行（`pnpm fix`）
6. 動作確認

## セキュリティ / 機密情報
- `data/config.json` などの設定ファイルに含まれる機密情報を Git にコミットしない
- ログに個人情報や認証情報を出力しない

## リポジトリ固有
- Steam API と SteamDB スクレイピングを組み合わせたツールであるため、外部サービスへの負荷に配慮する。
- 設定の変更時は `pnpm generate-schema` でスキーマを更新する。
